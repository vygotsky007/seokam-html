PDF 변환 방식
그림 모드는 페이지를 통째로 이미지로 넣습니다. HTML 재구성은 문항 나누기 결과를 문항별 HTML(발문·지문·선지)로 재구성하고, 변환이 안 된 문항만 조각 이미지로 넣습니다.
⚠ HTML 재구성을 쓰려면 먼저 [✂️ 문항 나누기] → [이대로 저장] 을 실행하세요. 그래야 문항별 HTML로 재구성됩니다.
PDF 답 입력칸(q1..qN)과 문항표에 함께 쓰입니다.
답칸을 자동으로 못 찾았어요. 문항 수에 맞춰 답칸 블록을 새로 넣을까요?

✂️ 문항 나누기 — 영역 조정

⚠ 빈 영역 있음
✏️ 직접 그리기 모드 — 페이지 위에서 드래그해 문항 영역을 그리세요. 그린 순서대로 번호가 붙습니다(번호 라벨 클릭으로 수정). Esc로 그리기 중단.
각 문항이 색 박스로 표시됩니다(번호는 시험지에 인쇄된 번호 그대로, 순서는 왼단 위→아래 → 오른단 위→아래). 번호가 잘못 읽혔으면 번호 라벨을 클릭해 고치세요. 박스 아래 손잡이(⇕)를 끌어 끝(=다음 문항 시작)을 조절하고, 박스를 클릭해 여러 개 선택 후 [선택 묶기]로 공통 지문 문항을 합치세요(페이지·단을 걸친 묶기는 [읽기 순서] 뷰에서 나란히 놓고 하면 쉽습니다). 선택 해제는 Esc, 실수는 Ctrl+Z로 되돌립니다. 빨간 점선은 빈 영역입니다.
🔍 조각 미리보기
문항 박스를 클릭하면 잘려 나올 조각이 여기 보여요.
